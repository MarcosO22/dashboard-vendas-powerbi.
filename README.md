# 📊 Dashboard Executivo | Análise de Supermercado

## 📝 Sobre o Projeto
Este dashboard foi desenvolvido para analisar o desempenho de vendas de uma rede varejista. O foco principal é a transformação de dados brutos em indicadores estratégicos (KPIs) para suporte à tomada de decisão.

## 🛠️ Tecnologias e Processos
* **Power BI:** Importação e tratamento de dados.
* **Modelagem:** Criação de uma tabela de metas personalizada para análise de performance (Realizado vs. Orçado).
* **DAX:** Desenvolvimento de medidas para Faturamento, Ticket Médio e metas.
* **UX/UI:** Design focado em Dark Mode para facilitar a leitura dos dados.

## 📈 KPIs Principais
* **Faturamento Total:** Acompanhamento em tempo real.
* **Ticket Médio:** Valor médio por venda.
* **Análise de Metas:** Comparativo visual através de gráfico de velocímetro (Gauge).
